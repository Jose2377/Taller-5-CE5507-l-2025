# Taller-5-CE5507-l-2025

a. Nombre del sistema: Cubo Interactivo 5x5x5.

b. Descripción general: Cubo conformado por luces led puestos en forma de cubo para mostrar la figura humana, todo controlado por un microcontrolador arduino.

c. Usuario objetivo: Estudiantes y Académicos, Desarrolladores de visualización de datos, Médicos, Enfermeros

d. Entradas: Botón de interrupción para cambiar modo de operación

e. Salidas: Matriz volumétrica con patrón específico
